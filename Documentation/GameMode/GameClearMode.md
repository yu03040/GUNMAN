# GameClearMode クラスの概要

ソースコード: `Source/GUNMAN/GameModes/GameClearMode.h / .cpp`

## 概要

`AGameClearMode` は `AGUNMANGameModeBase` を継承した**ゲームクリア画面専用の GameMode** です。  
現時点では独自のプロパティ・関数は持たないスタブクラスです。  
クリア画面の UI 操作・入力処理は `AGameClearMapScript`（LevelScript）が担当します。

クラス継承図は [GUNMANGameModeBase](GUNMANGameModeBase.md) を参照してください。
